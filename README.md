| Problem                      | Pattern                 | Real Example                   |
| ---------------------------- | ----------------------- | ------------------------------ |
| Only one instance            | Singleton               | Logger, Config                 |
| Creating objects             | Factory / Builder       | Payment Factory, Query Builder |
| Group of related objects     | Abstract Factory        | Stripe SDK Factory             |
| Add behavior dynamically     | Decorator               | Middleware                     |
| Convert interface            | Adapter                 | Legacy API wrapper             |
| Simplify system              | Facade                  | Laravel Facades                |
| Lazy load / access control   | Proxy                   | ORM lazy load                  |
| Notify subscribers           | Observer                | Events / Listeners             |
| Choose algorithm dynamically | Strategy                | Shipping / Payment             |
| Encapsulate actions          | Command                 | Laravel Jobs                   |
| Object behavior by state     | State                   | Order statuses                 |
| Pass through handlers        | Chain of Responsibility | Middleware chain               |
